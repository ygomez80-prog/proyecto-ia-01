# AGENT Project Overview

Este proyecto está organizado para trabajar con datos de ventas, prompts de agentes y resultados de análisis.

## Estructura principal

- `.agent/` : Contiene los skills o roles de prompt engineering usados en el proyecto.
- `Data/` : Contiene los archivos y documentos de trabajo con los datos de entrada.
- `Informes/` : Contiene los resultados y entregables generados por el análisis.

## Contenido de `.agent`

En `.agent` se encuentran los siguientes archivos de skill:

- `analista-datos-negocio.md` : Skill orientado a análisis de negocios y BI.
- `creador-contenido-ventas.md` : Skill orientado a copywriting y mensajes de ventas.
- `diseñador-clases-creativas.md` : Skill orientado a diseño instruccional y creación de clases.

## Contenido de `Data`

- `Data/raw/` : Datos de origen sin procesar para análisis y pruebas.
- `Data/ventas/` : Archivos relacionados con ventas e inventarios.

## Contenido de `Informes`

- Carpeta destinada a alojar los resultados, reportes y conclusiones obtenidas a partir del análisis de datos.

## Uso recomendado

1. Revisar los skills en `.agent` para entender los roles y enfoques disponibles.
2. Usar los archivos de `Data/` como base para la limpieza, el análisis y la generación de métricas.
3. Guardar los informes finales en `Informes/` para organizar los resultados obtenidos.
